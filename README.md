# Design-ReadMe-Formatter                                                           
                                                                                                  
  A browser-based Markdown editor for design students documenting their creative process. Built   
  for MDDN242 at Te Herenga Waka — Victoria University of Wellington.
                                                                                                  
  ## What it is                                                                                   
  
  A structured README editor with section templates tailored to design work — not software        
  documentation. Students pick sections (Design Intent, Ideation, AI & Prompting Process, etc.),
  fill in the prompts, and export clean Markdown ready to paste into their GitHub repo.           
                                                            
The sections here are about *how you designed something, not how to deploy it.
                                                                                                  
  ## Use it                                                                                       
  
  → [[GitHub Pages]](https://faultedstar.github.io/Design-ReadMe-Formatter/)                                                                  
                                                            
  No login, no account, no backend. Works entirely in the browser. Progress saves to localStorage.
                                                            
  ## Run locally                                                                                  
                                                            
  Just open `index.html`. No build step.
